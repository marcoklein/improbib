Improbib

A knowledge graph of improvisational theatre, derived from open web sources.

Improbib scrapes improv game databases, uses LLMs to extract structured game mechanics, skills, and how-to-play steps, then builds a cross-source, multilingual knowledge graph connecting elements, mechanics, skills, tags, and sources.
